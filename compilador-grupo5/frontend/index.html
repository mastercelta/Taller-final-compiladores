📘 Instrucciones de Gramática

- Declarar una variable:
  def nombreVariable igual expresión;

- Mostrar una variable o resultado:
  mostrar expresión;

- Operaciones soportadas:
  +  suma
  -  resta
  *  multiplicación
  /  división

- Puedes usar variables, números y paréntesis:
  def resultado igual (a + b) * 2;
  mostrar resultado;

🧪 Ejemplo completo:

def a igual 5;
def b igual 3;
def resultado igual a + b * 2;
mostrar resultado;
